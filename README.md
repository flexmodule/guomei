# guomei
用jquery配合require.js模块，以及sass预编译语言，进行模仿国美在线商城开发</br>
从数据库拿到的图片url地址进行数据模拟
-------------------------
不同的页面或者功能按照模块进行整合，实现首页（轮播图，楼梯，tab切换结合轮播图，以及置顶菜单），详情页（实现商品放大镜效果），购物车（推荐列表页的商品加入购物车，利用加入cookie的商品id进行真实模拟），登录，注册（表单验证，用到了随机数验证码）</br>
运行需要服务器环境，以及数据库</br>
